# 最最简单的skills教程，在TRAE无痛上手指南

![Cover Image](AI/assets/最最简单的skills教程，在TRAE无痛上手指南/file-20260120202524418.jpg)

1）很多小伙伴被 Prompt、Rule、Skills、MCP 绕晕了
	
Prompt (临时指令) ：像便利贴。用一次扔一张，下次还得重写，费嘴。
	
Rule (员工守则) ：像职业素养。AI 24 小时在脑子里背诵，规则多了就占脑容量（费 Token），反应变慢。
	
Skills (专业手册) ：像锁在抽屉里的 SOP。平时不占脑子，只有干这活时才翻开看。最省钱、最精准！
	
MCP (外接工具) ：像外挂插头。让 AI 能查天气、翻数据库，连接外部世界。
	
2）TRAE 无痛上手指南
	
不用写代码，两招让 AI 变身领域专家：
	
a. 让 AI 帮你写技能，直接对 TRAE 喊话：
	
“帮我在 ./trae/skills 下创建一个 Skill，名字叫‘小红书爆款文案官’。要求：分析我提供的素材，按钩子开头、痛点描述的格式输出。”
	
b. 拿来主义直接用， Awesome Claude Skills，把喜欢的文件夹下载下来，直接拖进 TRAE 的 【设置 -> 规则和技能 -> 技能】。

```
#trae# #ai# #AI# #大模型# #科技# #人工智能# #互联网大厂# #skills# #字节# #claude#
```
