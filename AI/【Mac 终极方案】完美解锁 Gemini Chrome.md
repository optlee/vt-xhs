# 【Mac 终极方案】完美解锁 Gemini Chrome

![Cover Image](AI/assets/【Mac%20终极方案】完美解锁%20Gemini%20Chrome/Mac-终极方案完美解锁-Gemini-Chrome-0-1770218599151.jpg)

第一步：升级最新版Chrome，开us梯子，规则模式即可
第二步：开启底层功能开关
* 打开 Chrome，在地址栏输入：chrome://flags
* 搜索框输入关键词：glic
* 将搜索到的所有相关选项改为enable
* 点击右下角的 Relaunch 重启浏览器
注：GLIC 即Gemini Live in Chrome
第三步：修改 macOS 系统级应用语言
在 Chrome 内部改语言没用。必须修改 macOS 分配给它的环境。
* 打开 Mac 的 系统设置 -> 通用 -> 语言与地区。
* 滑动到底部，找到 “应用程序” 版块。
* 点击 + 号，选择 Google Chrome。
* 将语言强制指定为 英语 (English)。
* 点击保存（此时请彻底关闭 Chrome）。
第四步：制作自动启动器
为了避免每次都要打开终端输命令，做一个App。
* 打开 Mac 自带的 “自动操作” (Automator) APP。
* 新建文档，选择 “应用程序” (Application)。
* 在左侧列表搜索 “运行 Shell 脚本”，双击添加。
* 清空 编辑框里的所有默认代码，粘贴下面这一行：
open -n -b com.google.Chrome --args --variations-override-country=us
* 按 Cmd + S 保存，命名为 “Chrome AI”，保存到“应用程序”文件夹中。
第五步：替换图标
默认生成的 App 图标是个机器人，替换成美观的 Chrome 图标。
准备工作：
一张高清 PNG 图片。
操作流程：
* 打开“访信” (Finder)，找到你刚才保存的 “Chrome AI” 应用程序。
* 右键点击它，选择 “显示简介” (Get Info)。
* 直接把 logo文件拖拽到简介窗口左上角那个小小的机器人图标上。
* 当鼠标指针旁出现绿色的 + 号时松手，图标即刻替换。
最终效果
以后请点击这个 Chrome AI 图标启动浏览器，仍然需要开梯子!!!!只是不用全局模式，规则模式就行。。你将获得：
✅ 无需全局代理，国内网站直连，AI 功能秒开。
✅ 地址栏出现 Gemini 快捷入口。
✅ 解锁 Glic 侧边栏悬浮窗体验
