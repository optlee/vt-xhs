# 开年甜宠‼️《花样男子》作者少女动漫神作

![Cover Image](http://sns-webpic-qc.xhscdn.com/202601210101/606518e5978b6c4bcc5b29cbaec6f5b9/spectrum/1040g34o31rc1q6i86u2g42l95mn80i68a74cmto!nd_dft_wlteh_jpg_3)

🌸霸权社 WIT STUDIO 出品
🌸《杀死比尔》导演执导｜20 话宝石级动画
🌸英伦青春 + 梦想竞技｜甜虐交织太好嗑
	
🌸【伦敦美术学院：半年夺冠的追梦赌约】🌸
20 世纪初伦敦，追梦少女一条院莉莉转入顶尖美术学院，与父母立下赌约：半年内拿下学院头筹，否则即刻回国。初来乍到的她，将大贵族出身、我行我素的天才学生基特・丘奇，视作终极对手。
	
🌸【宿敌变心动：天才与直球的双向吸引】🌸
原本痴迷绘画的基特，被莉莉直率纯粹的追梦执念打动；一心冲榜的莉莉，也在较量中看见基特不羁下的才华。预告藏满梦想与希望，主视觉图里伦敦街头的角色同框，满是恋爱预感，青春悸动在画笔间悄然蔓延。
	
🌸【神仙阵容：质感与颜值双在线】🌸
制作班底豪华拉满！《花样男子》神尾叶子亲自操刀原作与编剧，《杀死比尔》动画导演中泽一登执导，WIT STUDIO 负责制作。实力声优加盟，主题曲《star flower》治愈动听，全 20 话每集都如宝石般闪耀。
