# 为什么组织里叙事的重要性远大于实质

![Cover Image](../media/为什么组织里叙事的重要性远大于实质-0-1770006457783.jpg)

组织偏好叙事，不是文化问题，也不是管理层的问题； 而是因为：
真实太复杂、太慢、太不可控，系统无法直接处理。 因此：
决策依赖叙事
权力通过叙事运作
责任通过叙事重写
部门通过叙事对齐
评价体系通过叙事赋值
最终：
组织是叙事驱动的系统， 实质只是叙事的素材，而非决定力量。

```
#过渡环节的组织# #变革领导力# #矛盾的对立统一# #事物的本质# #系统性思维# #尊重与理解# #处理事情的能力# #事实和观点#
```
