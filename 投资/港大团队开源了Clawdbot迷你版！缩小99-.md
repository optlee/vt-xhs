# 港大团队开源了Clawdbot迷你版！缩小99%

![Cover Image](投资/assets/港大团队开源了Clawdbot迷你版！缩小99-/港大团队开源了Clawdbot迷你版缩小99-0-1770423907898.jpg)

受Clawdbot启发，香港大学数据科学实验室（HKUDS）开源了一个超轻量级个人AI助手nanobot，如果你觉得现在的Agent框架太重、代码太难读， nanobot 绝对会让你眼前一亮！
	
核心亮点：极致的轻量化
nanobot 的设计初衷是做一个“超轻量级”的个人 AI 助理。 它最令人惊叹的地方在于：
代码极简：其核心功能仅用了约 3,400 行代码 实现，比灵感来源 Clawdbot 缩减了 99%（后者超过 43 万行）。
开发友好：代码结构清晰易读，非常适合开发者进行二次开发、修改或用于学术研究。
性能强劲：更小的体量意味着更快的启动速度、更低的资源占用和更敏捷的迭代。
	
它能帮你做什么？
	
别看它“小”，功能却非常全能：
全能助理：支持 24/7 实时市场分析、全栈软件开发辅助以及个人知识库管理。
	
多端接入：你可以通过 Telegram、Discord、WhatsApp，甚至是最近刚更新支持的 飞书（Feishu） 与它对话。
	
多模型支持：不仅支持 OpenRouter 和 DeepSeek 等API 接入，还支持本地部署，保护隐私的同时保证响应速度。
	
智能任务调度：支持自然语言处理任务调度和定时任务，让 AI 帮你管理日常工作流。
	
快速上手：2 分钟部署
nanobot 的部署非常友好，几乎是“一键式”体验：
安装：直接通过 pip install nanobot-ai 即可安装稳定版。
配置：在配置文件中填入你的 API Key。
开聊：使用命令 nanobot agent -m "你好" 即可开启对话。
	
nanobot 证明了 AI 助理并不一定需要庞大的代码库。 这种“小而美”的架构不仅让普通用户能快速拥有一个全天候的数字助手，也为开发者提供了一个绝佳的实验平台。
	
目前该项目在 GitHub 上已获得超过 9.9k stars了，非常受欢迎。
